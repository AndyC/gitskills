git用户名和邮箱的作用：
用户名和邮箱地址相当于你的身份标识，是本地Git客户端的一个变量，不会随着Git库而改变。
每次commit都会有用户名和邮箱记录。
GitHub的contributions跟你的邮箱是关联的。
查看自己的用户名和邮箱地址：
git config user.name
git config user.email

修改自己的用户名和邮箱地址：
git config --global user.name "xxx"
git config --global user.email ""
